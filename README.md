# Insurance-claim-prediction
# Machine Learning project using PySpark to predict insurance claims

Project Description

This project focuses on predicting insurance claims using machine learning techniques implemented with PySpark. The dataset includes customer demographics, vehicle specifications, and policy-related attributes, which are processed using Apache Spark for efficient handling of large-scale data.

The project involves data preprocessing, feature engineering, and handling class imbalance to improve model performance. Machine learning models such as Logistic Regression and Random Forest are used to classify whether a customer is likely to make an insurance claim.

Additionally, exploratory data analysis and visualizations are performed to understand how different factors like fuel type, transmission type, and customer age influence claim behavior. The results demonstrate that scalable data processing combined with machine learning can effectively support decision-making in the insurance industry.
